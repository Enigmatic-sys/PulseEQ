### Minimalist 12-Band Parametric Equalizer for Chromium

PulseEQ is a high-performance, audio engine designed for users who crave precision without the clutter. Built on Linux with a focus on **Glassmorphism**, **Privacy**, and **Audio Fidelity**.

---

## üöÄ Key Features

* **12-Band Precision:** Optimized UI that fits a full 12-band rack on one screen‚Äîno scrolling required.
* **True Parametric Control:** Manually adjust **Frequency (Hz)** and **Q-Factor (Bandwidth)** for surgical audio editing.
* **iOS-Style Glassmorphism:** A borderless, "edge-to-edge" frosted glass interface with dynamic mesh gradients.
* **Intelligent Visual Feedback:** dB counters shift from **Cool Blue** to **Peak Red** as gain increases, providing a
    professional visual warning for digital clipping.
* **Custom Preset Engine:** Bass Max, Rock, Pop, Jazz, and Electronic. You can create your own preset.
* **User Profiles:** Save, name, and manage your own infinite library of sound signatures.
* **Privacy First:** No tracking, no data collection, no background scripts. Pure Web Audio API implementation.

## üõ† Installation (Manual Sideload)

Since PulseEQ is currently an independent project (saving up for that $5 Chrome Web Store fee!), you can install it manually in 30 seconds:

1.  **Download:** Click the green **Code** button and select **Download ZIP**.
2.  **Extract:** Unzip the folder to a location you won't delete (like a "Documents/Extensions" folder).
3.  **Extensions Page:** Open your browser and go to `chrome://extensions` (or `brave://extensions`).
4.  **Developer Mode:** Toggle the **Developer Mode** switch in the top-right corner.
5.  **Load Unpacked:** Click the **Load unpacked** button and select the `MyEQExtension` folder.
6.  **Pin it:** Click the puzzle icon üß© in your toolbar and pin PulseEQ for instant access.

## üéõ How to Use

1.  Start playing audio in any tab (YouTube, Spotify, Soundcloud, etc).
